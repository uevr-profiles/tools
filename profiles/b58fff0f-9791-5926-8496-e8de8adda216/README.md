## Description

3dof weapons working very well (including looking through scopes!). Just had to tweak VR_CameraRightOffset to -3.3 and barely tweaked VR_CameraUpOffset to 0.2. In the game's input settings I had to change the forward/backward/left/right buttons to WSAD keys. Synched Sequential ended up working best + ghosting fix.  Lone Wolf missions are incredible in VR. 

EDIT: I did some testing and confirmed that you need to install Mutar's plugin from https://github.com/mutars/uevr-software-gamepad-plugin. Install VigEm bus using the installer first, and then copy viGemBridge.dll into the plugins folder of the UEVR profile after installing ViGEM bus. This is the easiest way to get controls working. Verified as working today. You can add that viGemBridge.dll to any other game profile as well is you experience a similar issue.
