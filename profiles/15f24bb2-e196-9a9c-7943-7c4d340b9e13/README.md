## Description

**-- 6DOF profile for Lefties --**
Below is my custom profile for left-handed people. 
This builds on Tyriel's excellent 6DOF profile, details for what that does is here:
https://discord.com/channels/747967102895390741/1093869639714484244/1192496331659431936

**Changes in my version:**
1. Gun moved to left hand
2. Swapped left and right triggers
3. Changed the X and B button swap to be on XInput level instead of OpenXR, so should work with any controller and OpenVR as well
4. Switched DPad back to right-thumbrest-left-stick
5. A few further CVAR tweaks to improve image quality (subjective, yymv)

Dev'd and tested on Quest 3, but should work with any HMD and controller set. I don't have any other to test though.
