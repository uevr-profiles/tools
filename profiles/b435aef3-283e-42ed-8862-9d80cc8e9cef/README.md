6DOF

Aiming with the fire hose nozzle is working. I had to attach the nozzle + line up the actual water stream to the right controller. The main thing left to do is to hide the rear hose or attach it to the right hand. However, after lots of hunting around I couldn't find the hose mesh. So you will see the hose awkwardly disconnected at the rear (which should be an easy fix when someone finds that mesh). This game is actually incredible in VR with very nice visuals (I set to DLSS Performance). Honestly, the controls work great AS IS.
