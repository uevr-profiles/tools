## Description

6DOF Profile

Notes:
- Flashlight, Self-generating flashlight and Sound level meter are attached to the Left Controller.
- Interaction items are attached to the Right Controller. 

Press Left-Ctrl key to Enable/Disable the 6DOF and camera object hook for cutscenes.

Known issues:
- If you have activated the 6DOF object's hook, an error message will appear when loading a save or a new chapter, requiring a restart of the game.
