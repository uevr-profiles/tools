## Description

Here is a motion controller profile. It lets you control the cursor with the right motion controller. It also has a script to enable auto sprint anytime you walk forward max amount (holding LB was annoying).

Also, adds the fish eye cvars.  I'd recommend that you get dlss swapper and swap in the latest DLSS library and enable profile J in nvidia inspector.
