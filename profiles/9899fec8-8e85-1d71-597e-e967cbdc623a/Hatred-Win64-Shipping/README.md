## Description

heres my config, fixed SMG orientation as well as reenabled notims body visibility because i find the hands help let you know where youre facing as well as makes executions not feel like a ghost is performing them.

For those with **problems turning**, I found a solution:
1. Make sure "Aiming sensitivity reduction" in Controls+ is turned to NONE
2. make sure there is an extra gamepad plugged in while you are playing, idk why but it works if youre unable to turn more than a mm/sec
3. Find Game.ini and change "GamepadSensitivityScale" to something like 1.5 or more.
