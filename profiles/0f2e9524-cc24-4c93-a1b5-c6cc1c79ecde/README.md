## Description

Motion Gestures 6DOF Profile based on TimBurton's Profile

Instructions:
Needs the latest nightly 1040+:https://github.com/praydog/UEVR-nightly/releases/latest
Import and Inject before or as soon as you see the claptrap loading screen
Advised to turn off autoaim in settings for best results.
Have fun.

New Features: 
Holster System:
-RGrip+ Shoulders or Rhip or CenterLower Back : Weaponslots 1-4
-RGrip+ RIght Chest:Grab Grenade, Release RGrip to throw
-LGrip+ LShoulder: INventory .       +LHip : Map
Weapon interaction Zones:
-LGrip below gun: Reloading
-LTrigger right above Right Controller: Switch fire mode
Melee: Swing your right hand really fast, and try to stop when crosshair is on enemy for best result (sensitivity can be tweaked in melee lua)
Rebinding:
-Rstick up for jump, 
-Rstick down for Crouch, 
-LGrip for Sprint, you can still sprint with the poor LThumbButton
-A Button for special ability
