## Description

Tested this with Quest 3 & Virtual Desktop VDXR  
Runs well in native stereo

Store page doesn't mention controller support, but once the gameplay starts the Quest controls work with these buttons - doesn't seem remappable in options - it shows kbd and mouse controls

Shoot: RT
Melee: B
Crouch: RG
Throw Hat: LT
Switch Weapon Mode: Y
Jump: A
Dash LG

I also changed control method to right controller for 3DOF

Anytime you die or get to a tutorial part, have to switch back to desktop view in Virtual Desktop to use the mouse to select options.


Subtitles seem to appear way below your view so you can't read them

The gun model and FPS hands are far in front of your view compared to flat version, looking to the side the models seem to warp a bit, but not as bad as Anacrusis and Bears in Space that I played.

Hiding the FPS hands will also hide the melee animation.  I did move the camera back a little so it's closer to the player view but then there's some clipping in the gun.

Fun little game from what I played.  

Worth checking out since the game is free.

Attaching my profile if anyone wants to mess with it and see if anything can be done about the FPS hand and gun models.
