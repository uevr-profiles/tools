## Description

Upon further research, it seems that it *should* be possible to map buttons to the Index "System" button, 
But it apparently depends on some other factors like which Runtime is being used (SteamVR or OpenXR), etc..
It should be possible tho, to bind "Tab" to your track-pad.

With that said however, I know rebinding buttons isn't the most elegant solution (especially when using Steam)
So I have made a slight change to the mod and have now included a binding to Open your D.I.D
(Also I don't have Index Controllers so not really able to do any meaningful tests)

I have tied it too the "Status" screen, ..
So that when you have the "Status" screen open allowing you to select a weapon or torch (using your Right Grip), 
you can now Press your "Left Grip" to open/activate your D.I.D
I stress tested this earlier and it does not interfere with any movement (like jogging)

So to clarify, ... 
- Press your "Right Grip" to open the "Status" screen
- Press your "Left Grip" while the Status screen is open to activate your D.I.D

I have attached the profile with the modifications (everything else is exactly the same).
Let me know if this works for you.
