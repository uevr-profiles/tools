## Description

I can't claim any fancy scripting here or anything, but here's my profile, I still need to joystick around to point at things but it looks good and plays well, once you get the hang of the mechanics. Decoupled pitch is enabled. I did notice that sometimes it breaks and the camera starts swinging around, but toggling decoupled fixes it again. The only other thing I really did was switch the D-Pad to left thumbrest, right stick, only because it matches Dead Island 2 settings and I was pretty used to that 🙂
