## Description

6DOF.

- Profile camera for infantry, vehicles and only few mounted weapons like the dhska.
- When you mount on a vehicle open the UEVR U.I , select "Load Camera 1", under "input" change "Aim Method" and  "Movement Orientation"  from  "Head/HMD" to "Game" and deselect the box "Roomscale Movment".
- For the mounted weapon it's the same as for the vehicles except you need to use the "Load camera 2".
- When you want to go back to infantry, you need to reselect the box "Roomscale Movement "  , Load "Camera0" and change  "Aim Method " and  "Movement Orientation"  from  "Game" to " Head/HMD ".

Note: Some object like binoculars are not working properly. 


input.ini file (keybinds) for VR Squad on Quest 3.

Put the file in "%localappdata%\SquadGame\Saved\Config\WindowsNoEditor"
