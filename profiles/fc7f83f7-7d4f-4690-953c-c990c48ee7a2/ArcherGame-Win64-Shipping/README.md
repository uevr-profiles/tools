## Description

I wrote a plugin included in the profile so that you can aim and fire with the right controller. 

- The right stick moves the mouse pointer.
- The left A button (or B on gamepad)  brings up the menu.
- The left stick up = 1
- Left stick right = 2
- Left stick down = 3
- Right button = zoom to aim (uses camera0 and camera1 in profile)
- Right A button moves to next tower.
