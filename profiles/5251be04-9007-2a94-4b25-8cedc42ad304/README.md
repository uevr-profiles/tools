## Description

**Custom Profile with 3 Camera Views:**

Normal-view, Medium-view, first-person horse/Celtacea and first-person (player). All views have been set to keyboard hotkeys. **To begin make sure within the UEVR menu/Runtime to hit Recenter View, so your view becomes full center.**

Normal-view is set to 'A', medium-view is set to 'S', first-person horse/Celtacea is set to 'D' and first-person (player) is set to 'spacebar'.  First-person horse has to be combined with first-person (player). More on that below.

**If First-person View Stops Working:**

Most likely when you close out of the game and restart, the first-person view will become inactive. In order to re-activate it you just have to go to config under UObjecthook, and click 'Enabled at Startup'. If you can't see the config option select 'Show Advanced Options' in the upper-left corner of the UEVR menu.

**Viewing Cutscenes Correctly:**

Whenever there is a story cutscene switch back to normal view by pressing 'A'. This ensures everything will be in full view. 

**When Ridding the Horse and Flying with Cetacea:**

When you get on the horse or later, flying with Celtacea, press 'D' to set up the view, then press 'Spacebar' to go into first-person.

**Medium-view for Battles and Exploration:**

Both normal-view and medium-view are good for battles. With medium-view getting you closer up to the characters. Medium-view is also good for map/field exploration as you can keep this view on for both exploration and battles without having to constantly switch back and forth. 

**First-person View in Towns and for the Ship:**

First-person view is awesome in towns, the field and the Ship.  Before initiating first-person view on foot or for the Ship, always make sure you are in normal-view 'A' first as the other views aren't set  height-wise to accommodate it.  But once you get into battle you will be too zoomed in. So, I recommend medium-view for both battles and exploration as mentioned above.
