## Description

I’ve made some adjustments to a profile provided by others since I couldn’t find a fully 100% working profile for a comfortable gameplay experience.
Once again – THIS IS NOT 100% MY PROFILE, BUT IT HAS BEEN COMPLETELY OVERHAULED BY ME!

Here’s what I managed to fix (even if some things required workarounds):
 • The character is visible in cutscenes and during executions. This is achieved by adjusting the camera, so the freely moving hands don’t interfere. And if they do slightly get in the way (they’re a little visible near the chest), you can move them down with the stick and they’ll be completely out of sight.

 • Fixed the gun that was sticking sideways at the level with the sewers, and also further along in the game when it appeared again.

 • Adjusted the flashlight on the gun so it doesn’t glare or blind you.

 • Tweaked the crosshair. Not perfect, but better than before. If it shifts during gameplay, reset the view using the controller button – it shouldn’t drift too much.

 • Fixed the turrets on SWAT and army vehicles. This was done with yet another workaround. The turret actually hangs over the entire level, with the barrel sticking out of your forehead 😅, but you can’t see it because it’s hidden. This allows you to freely control the turret when you’re in it on a vehicle.

 • Reconfigured the buttons so everything works conveniently. Tested on my Oculus Meta Quest 3. Weapons are selected using the DPAD, meaning you can rest your thumb on the edge of the right Oculus controller, and the stick on the left controller switches to weapon selection mode instead of running. Left-right-up selects firearms, while down on the stick selects grenades. Running is assigned to the left trigger, and target selection to the left bumper, etc.

The only thing I haven’t managed yet, but would like to, is MOVE THE SUBTITLES HIGHER SO THEY ARE CLEARLY VISIBLE IN VR. If anyone knows how to do this, I’d really appreciate the help!
