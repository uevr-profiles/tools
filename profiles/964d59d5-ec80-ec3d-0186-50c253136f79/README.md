## Description

**6DOF Motion Control Profile** *(Work in Progress)*

**Features**
- Both arms work
- Items attach to the **right arm**

**Known Issues**
1. You must use the **Adjust Tool** on every launch for the **right arm**
   - Takes only a few seconds — see the **pinned tutorial**
2. When using **gravity platforms**, the angle may be incorrect
3. **Finger movement is not implemented yet** (planned for a future update)
