WIP 6dof all weapons (early work) 
Buttons remapped (to remove button remap and swing left controller melee delete script and plugin folder)

LB+R3= Disable UObject
LB+A= Weapon wheel (dpad quick slots)
LB+RstickUp= Zoom in (help with sniping "not really")
LB+RstickDown= Zoom back
LeftcontrollerMenuButton= Map
LeftcontrollerMenuButton(hold)= Pause
A= Interact
B= Crouch
X= 360 turn
LT= Melee (also swinging  Lcontroller you can melee, swing can be adjusted in Script UI and if swing not working reload plugin)
RT= Fire
RB= Aim
Hold both controllers together= Reload

Melee and Healing still see silhouette
Climbing ladders, pulling carts or boats have a delay (swing right controller up and down) also sidling\ledge
