## Description

So, this is my personal profile for Deep Rock Galactic using the Early December 2025 nightly build, it's keyboard and mouse controls but the up and down aim is independent of which way you're looking which is nice cause it makes you feel like your head is separate from your hands, which gives it more of a vr game feeling while still having the convenience of keyboard and mouse. I'm not super knowledgeable about UEVR, I'm sure there are ways to make it look better, but this feels pleasant enough and playable.
