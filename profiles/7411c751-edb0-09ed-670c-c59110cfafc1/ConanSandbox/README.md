## Description

So, I've got it back up how I had it. I think this is how far I can go using just UEVR. So, I play in 1st person view, so all I've set up is based on that. I don't think it will fit very well for 3rd person. 

My setup is that I attached the HUD to right controller, so that you can flip it up/down as you like, it's not up in your face all the time. Remember that the target is also in the middle of the HUD, so your character will face wherever your target is facing. When I walk normally in game, I just have my right wrist slightly tilted to the left, so that my character is faced forward. 

Further I've scaled the HUD down quite a lot and moved it closer to the screen, it works better with the right controller bound setup imo. 

I've also activated snap turn and set it to 30. 

I'll upload my profile right away, hope you like it, and just tweak away whatever you don't 😊
