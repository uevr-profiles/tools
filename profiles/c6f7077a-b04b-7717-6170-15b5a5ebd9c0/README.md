## Description

This is a modified version of letmein's profile. I just changed the HUD so that it follows your view and also moved it closer towards the center so the edges are easier to read. As this is such a fast paced game you need to keep an eye on your health/ammo/weapon etc. which is hard to do when it's shaking on your wrist.
