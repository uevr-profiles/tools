## Description

It's not quite full motion controls.
- You can aim with your controller at what you want to interact with, then hit the top right button and it will harvest, or click the trigger and it will swing the ax.
- You can use the dpad to switch between tools on your hot bar. Hold down the left system button very briefly for the game menu or for a couple seconds for the crafting menu.
- If you have your cursor visible (there's a setting in UEVR to toggle cursor display always on), then you will need to use a mouse to navigate the menu
