## Description

- 6 DOF aim right controller. Trimmed down hands a bit so 6dof works
- Plugin that fixes the talents menu so you can enter and exit the menu without having to bring up UEVR menu and toggle follow UI
- There are some personal preference stuff baked in. 1 is the size and distance of the menu is part of the plugin. The distance is maxed so that the cursor is not doubled up when aiming. The size is large in all cases but automatically is shrinked when in the talents menu since follow UI is on and you can only see edges with your eyes in this case.
- I added some user_script.txt entries to disable some things like motion blur and other things.
- In this profile, to bring up UEVR menu you must hold L3+R3. This is because there's an in-game move using these two buttons together.

Note:
- It may be a little funky at the very beginning of the game when using the menus before they are all unlocked. If anything weird happens at any point with the aim just enter and exit the menu right away and it should be back to normal.
