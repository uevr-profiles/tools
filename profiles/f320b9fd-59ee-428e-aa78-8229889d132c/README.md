## Description

1st Person Mod.

If using Head-Aiming:

- Disable gas-mask "Acknowledged Pawn/Components/StaticMeshComponent GasMask"
- Set "NearClipPlane" to "19.0" to make the head invisible (increase or decrease to your own liking)

If you have difficulty seeing pad-locks, etc:
- Disable Roomscale

If you wish to use motion controllers, etc:
- Turn off visibility to SkeletalCharacterMesh.

Place the .pak file in your Daymare 1994 pak folder ("Daymare 1994 Sandcastle\Ogre\Content\Paks") .
