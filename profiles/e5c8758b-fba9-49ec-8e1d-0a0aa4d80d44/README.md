## Description

I've set the main camera & uObject hooks all to the same hotkey [ U ], this allows you to toggle between 3rd person and 1st person with the player mesh hidden at the touch of a button (remap a controller binding to U or update this to your key of choice, etc)

Likewise, I've set the reset-camera position to another key [ K ], you do need to push the default attach point down a ways to get a better FPV but that'll cause you to clip into the ground during cutscenes, so toggle back to 3rd person and reset cam view to get the best experience for watching the in game scenes.
