## Description

Updated 1st Person Mod

If using Head-Aiming:
- Set "NearClipPlane" to "13.0" to make the head invisible (increase or decrease to your own liking)
- Disable "SpotLightComponent TorchLight/Children/StaticMeshComponent godray_cone_02" (stops NearClip affecting  torch)

If you have difficulty seeing DID device, disable Roomscale

If you wish to use motion controllers (3DOF Only):
- Turn off visibility to SkeletalCharacterMesh

Place the .pak file in your Daymare pak folder ("Daymare 1998\Daymare_MASTER\Content\Paks")
