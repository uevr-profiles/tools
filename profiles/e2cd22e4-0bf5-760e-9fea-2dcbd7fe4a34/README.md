## Description

Hi all,
Here's something a little bit different for you all to try out - a 1st person profile for this top down twin stick shooter game... 😎 

You have:
- Custom 1st person VR movement system with tweakable walk and turn speed values
- 6dof controller accurate shooting (tweaked all the guns I found in the tutorial)
- Toggle between 1st person and top down at any time with dpad right

**Known issues:**
- The sky box is messed up - so try to not look up 🤦‍♂️
- The lock sequence QTE workaround doesn't function the first time in the Garage gate lock in Rengkok south city (1st mission in the city after you talk to Rustam at the bar) - so if you fail this sequence the first time, you will be yeeted and the game will crash - so either try to succeed or do it in top down view mode. If you fail and reload the level, the lock sequence buttons might not show for you. so if that happens - go to UEVR UI - lua loader - unchek and then check "ruiner_vr.lua" and then engage the gate again and it should be fine.
- Going back to main menu in 1st person when in a non-combat level will crash the game - workaround - switch to "top sown" view with dpad right before quiting to main menu. 
- Picking up a different weapon in top down view then the one that was equipped in 1st person and then toggling back to 1st person will mess up the attached weapon.
- The bullets flashhider trace will sometime show in a weird angle.

But saying all that, what I played so far was a great fun. This is a real transformative way to play this game and it's very cheap right now so defintely recommend giving it a try 😉
