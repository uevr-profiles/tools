## Description

**3DOF profile**
Simple profile with 3DOF aiming how I like to play it with a script for small fixes.
- switches to game aiming method and turns off roomscale during character animations
- switches right thumbstick when Y is pressed, so that you can select tools

Issues:
- Animated parts where you're fully looking up o down. I'm not far in the game, but early on you are on your back looking at the sky and this was really awkward. You can also disable decoupled pitch during animations if you want (its commented out in the script), but I don't like that.
- It would be nice to switch to game aiming during the clue connecting sections but its perfectly playable as-is too.
- The main menu does some weird stuff in one eye with native fix on, but its fine when you get ingame and the fix is needed for some scenes
