# 6dof profile V1.06 for  Incursion V1.3

## [changelog:](https://discord.com/channels/747967102895390741/1204245050205143050/1476687239051214991)

Fixed reloading after using DPAD weapon switch 

## Features:
- hands
- recoil
- Holsters
- Colliison based reloading
- Scope PiP
- Motion Mouse Emulation in Menus

## Install: 
- use latest nightly

## Buttons:
- A: sprint
- B: Firemode
- RStick Up: jump
- RStick Down: Crouch
- DPAD: Key1-4, Weapons and GPS or use holsters
- Y: Use/Interact, you need to hold RGrip to aim the interaction
- Left Hand Trigger at Magazin: Reload or Mag Check
- hold both Left Grip(first) + LeftTrigger: to go into ADS mode for less recoil
- Mouse Emulation 
  - right controller rotation in menus
  - RightStick: works like mouse in Tablet
  - Press R Grip to Reset Mouse in Menus

## Holsters:
- Right Shoulder: Primary
- Right Hip: Secondary
- Left Shoulder: 2nd Primary
- Right Chest: Tablet

## Issues:
- with all the hands stuff there´s a slight memeory leak due to UE5.6, you should probably be fine at 32 gb ram to run at least 2-3 long missions, but check back your ram after every mission.
- when Holding RGrip for interaction aiming, you cannot aim the gun normally, but goes back to  normal once release rGrip
