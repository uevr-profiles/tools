## Description

Long pressing L3 and R3 enables/disables UObjecthook for first person.
But this also enables decoupled pitch with stays on after disabling UObjecthook, and this is a problem for some scenes (rocket launch is the first, and then the zero-g scenes).
However, from starting Mars on it works great, disable UObjecthook for cutscenes or when controlling the robot, enable during normal play.
If your controllers are not working, that is an issue of the Epic Games overlay.
