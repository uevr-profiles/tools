## Description

Hi All,
I made a new plugin for this game.

**Plugin details:**
- "1st person immersion mode" (1st person camera interaction icons are displayed only when an object can be interacted with for maximum immersion)
- Introduced "Enhanced Movement" system -  with tweak-able "speed" values (in UEVR UI - "Song of Horror VR" tab) 
- Functional hands with 6dof controller tracking
- Dpad right to toggle between "1st person immersion mode" and "3rd person fixed cameras investigation mode" at any time (to help in cases you think you missed an item/object to interact with in 1st person immersion mode) 
- Custom support for all the game mechanics I encountered so far (currently in middle of episode 1)
- Eliminated pop in via cvars
-  Changes to controls: map moved to right thumbstick. switching lightsource on/off is done only with right trigger.
- Quick recenter using both grab buttons at once

**Tips and Tricks:**
- The interaction icon sometimes may not appear/clip through meshes - Try to press "A" when you think something should be interacted with - also it is advised to toggle "3rd person fixed cameras investigation mode" every once in a while to see interaction icons for everything in the current room, to make sure you didn;t miss anything. In "3rd person fixed cameras investigation mode", the interaction icons will be displayed as diamods, white diamonds are for an object that you didn't interact with yet, whereas gray diamonds are for objects you already interacted with.
- Some interactable objects that are not on the character's eye level can only be interacted by lowering/raising where the character looks. this is done with right stick up/down - you can switch to  "3rd person fixed cameras investigation mode" to see where the character is looking currently.
