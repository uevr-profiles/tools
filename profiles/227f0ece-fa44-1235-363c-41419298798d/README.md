## Description

https://store.steampowered.com/app/2701660/DRAGON_QUEST_III_HD2D_Remake/
Game works perfectly. Performance is outstanding. I was able to increase resolution to 1.2 and run everything max settings. Could probably even go higher. Also, game is DX11 so openxr toolkit could be used.  Native rendering. Game froze when injecting on the opening splash screens the first time. Injecting at the menu works perfectly.

I made 3 cameras to toggle between in the included profile cranking up the world scale in the 2nd and 3rd much like Octopath Traveler. This works well in this game.  

Included profile is entirely optional. Game works great with all defaults. If you don't use a profile, make sure you go to cvars and disable all motion blur and depth of field.
