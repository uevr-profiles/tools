## Description

New version with various bug fixes:
- When starting to climb a ladder, the 1st person camera will switch immediately to the current fixed camera to make the transition less jarring.
- Hands will now re-appear automatically in case your controllers are waking up from sleep (instead of having to reload save to make them appear again).
- The weapon angle is fixed when transitioning to fixed cameras for the "Elevator Passage" room
