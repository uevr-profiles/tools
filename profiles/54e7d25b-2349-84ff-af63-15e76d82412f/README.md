## Features:
- hands
- recoil
- Holsters
- Colliison based reloading
- Scope PiP
- Motion Mouse Emulation in Menus

## Install: 
- use latest nightly

## Buttons:
A: sprint
B: Firemode
RStick Up: jump
RStick Down: Crouch
DPAD: Key1-4, Weapons and GPS or use holsters

Y: Use/Interact, you need to hold RGrip to aim the interaction

RightStick: works like mouse in Tablet

Left Hand Trigger at Magazin: Reload or Mag Check

hold both Left Grip(first) + LeftTrigger: to go into ADS mode for less recoil

Press R Grip to Reset Mouse in Menus

## Holsters:
- Right Shoulder: Primary
- Right Hip: Secondary
- Left Shoulder: 2nd Primary
- Right Chest: Tablet

## Issues:
