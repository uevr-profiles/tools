## Description

Roboquest profile that fixed the aiming that now follows the crosshair based on wanderyonder !
Also adjusted a few QOL things like snap turning, attach lerp, etc.

- Go to Roboquest game settings > Controls > Gamepad Magnetism and set that to 0 to prevent screen shaking while aiming.
- If you find that you are shooting below the crosshair, go to Settings in the actual RoboQuest game --> Game --> Find the Crosshair settings --> Turn on Recenter Crosshair"

Works in multiplayer.

Steam and GamePass Profiles.
