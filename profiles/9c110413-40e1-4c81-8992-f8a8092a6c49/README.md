## Description

Note: Contains the fix for sprinting in 3rd person.


1st Person Mod + Crosshair Mod + 3rd Person Toggle - WIP

[1st Person Mod]

- All cameras including DLC
- 1st Person enabled by default
- Includes 3rd Person Toggle (HOME)
- Auto toggle Head when switching POV
- Toggle Head On/Off (DEL)

[Crosshair Mod]

- Simple dot when not aiming, crosshair more visible when aiming
- Enabled by default toggle On/Off with "End" key
- Includes UE4SS (Setup ready to go)

Key-Binds:

HOME = Toggle 3rd/1st person
DEL  = Toggle Head On/Off
END  = Toggle Crosshair On/Off
CTRL = Enable UEVR mouse (if needed for menus)

Mod Install Instructions:

- Download and unzip "Remnant2_FP+CrosshairMod.zip"
- Drop the "Remnant2" folder into your main "Remnant2" folder

Note:
This is still a W.I.P and has some bugs that are being worked on:

- There is currently no Head or Motion Controller Aiming.
- When switching from 1st to 3rd (and visa versa) you need to initiate the camera by pressing melee/crouch/aim/sprint etc.
- You will still need to toggle your characters head for Cut-Scenes.
