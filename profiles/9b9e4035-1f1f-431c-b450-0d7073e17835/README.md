## Description

Head-Aiming UEVR Profile:
Modified TimBurtons Borderlands3 Fix
- Set up for Head-Aiming
- Character body/arms are visible

Aimed CrossHair Mod V3:
Works for both Head-Aiming and 6DOF if you attach the UI to your motion controller.

- Smaller crosshair will appear when you aim
- Is the same for ALL weapons
- Uses UML, "F1" to open UML UI
- The mod will remember your aiming mode
- Enabled by default, use Hot-Keys and/or UML UI to disable/re-enable

Key-Binds:
- (Home) Aim On
- (Left ALT) OFF

UML (Unreal Mod Loader): https://github.com/RussellJerome/UnrealModLoader/releases

1 - Place "BL3Crossmod.pak" in to folder named "LogicMods" in your Borderlands 3 pak folder (create if it doesn't exist) :
("Borderlands 3\OakGame\Content\Paks\LogicMods")

2 - Place the file "Borderlands3.profile" into your UnrealModLoader "Profiles" folder :
("UnrealModLoader_V2.2.1\Profiles")

3 - Start UML then Load game and smaller crosshair will appear when aiming
