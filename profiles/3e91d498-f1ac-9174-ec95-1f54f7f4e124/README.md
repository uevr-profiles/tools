## Description

# **FIRST PERSON PROFILE** V1.02 

**LATEST FEATURES: **
v1.02
- Added zooming to scopes (move right stick forward/backward when scope is active)
- Added brightness adjustment to scope (hold left dpad while moving right stick forward/backward when scope is active)
- Added mitigation for Gamma difference in left and right eye using Gamma Fix config setting (thanks to letmein50)
    None - No fix
    Standard - Fixes many issues with no loss of shadows
    Max - Fixes most issues but with loss of shadows
- Added three punching modes
    No Physical Punch - only right grab with throw punches
    Physical Punch + Right Grip Punch - Both physical punching and right grab will throw punches
    Physical Punch Only - Only physical punching will throw punches
