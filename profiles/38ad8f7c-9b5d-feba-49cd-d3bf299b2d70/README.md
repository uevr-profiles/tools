## Description

Here is a 6dof profile with first person, body invisible, hud and aiming attached to right controller, only revolver is tested, and shots are not 100% accurate, but hopefully the other weapons follow the attched object, if not, might need to find and attach the rest of the weapons.  Native stereo, not too much needed to get it looking good in VR, some game menus are still in Portuguese, so you might need google translate nearby for graphic settings.  There can be a little motion sickness since the character is in 3rd person and moves a little with momentum.  There is no scripting to detect when driving, so I set the 3 uevr cameras for = On foot is Camera 0, riding bikes load camera 1, in cars load camera 2.
