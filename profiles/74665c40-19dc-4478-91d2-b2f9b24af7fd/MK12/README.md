## Description

Keybinds:

7,8,9 is to recenter play space.

0 is normal camera.
1 is world scale @ 10 camera. 
2 is world scale @ 5, with decouple pitch, for when you want to align the game with the floor.

Put the GameUserSettings.ini file in "%localappdata%\MK12\Saved\Steam\Config\WindowsNoEditor".
