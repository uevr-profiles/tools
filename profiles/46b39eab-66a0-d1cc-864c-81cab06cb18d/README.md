## Description

6DOF profile native stereo, runs great and looks really good.  I didn't have any issues with the menus like zlink, maybe one of their updates fixed that.  I only have the first pistol attached, so I'm not sure how any other guns hook, but the machete is not lined up right.  I only found the one hook for weapon, so the position/rotation works on both machete and pistol, if you adjust one it adjusts the other, so one will be off.  Aiming works with the in game cursor and gun is accurate even without using the aim button.  If you would rather use 3dof aiming which works really well in this, just delete the weapon file in uobjecthook folder, the _mc_state file.  Adjust reolution if you need to lower it, my gpu is 5090 so I have it at 1.2.  Set gamma in cvars if it gets reset because it's washed out and too bright if left at 0.  I also bumped sharpening a bit since AA makes it a tiny bit blurry.
