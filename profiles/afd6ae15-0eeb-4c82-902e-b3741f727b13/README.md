## Description

Head-Aiming:
- Changed Runtime to OpenVR as game sometimes crashes with OpenXR
- Lowered camera a touch
- Increased world scale to be a bit more life-like

If when playing, your head-set suddenly looks odd, simply open the UEVR main window and click "Reinitialize Runtime". Then your view will return to normal.

Aim Camera Fix + Crosshair:
When aiming in this game there is no crosshair and the gun is too high for me.

Crosshair Mod:
- The mod will remember your aiming mode
- Crosshair appears when you aim
- Is the same for ALL guns
- Includes UE4SS

Aim Only enabled by default, use Hot-Keys change.

3 options, each with its own key-bind:
- (END) Always ON [This will place a Dot when not aiming and a Crosshair when aiming]
- (Left ALT) OFF
- (Home) Aim Only [This will enable a Crosshair only when aiming]

1 - Download "ChernobyliteCrosshairMod.zip" and unpack

2 - Place folder named "LogicMods" in your ChernobylGame pak folder :
("Chernobylite\ChernobylGame\Content\Paks\LogicMods")

3 - Place folder named "Win64" in your ChernobylGame Binaries folder :
("Chernobylite\ChernobylGame\Binaries\Win64")
