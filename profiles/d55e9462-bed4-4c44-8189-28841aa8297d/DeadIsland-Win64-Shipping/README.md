## Description

6DOF Profile.

- Virtual gunstock (two handed handling) added to ranged guns, including handguns. Hold left grip to activate it. Releasing it resets weapon position. One handed shooting should work as normal.
- Left shoulder button set to left trigger. Press left trigger one time to evade and hold it to throw melee weapons.
- Swing to attack exclusively with melee weapons. (Swing triggers right trigger button, not proper melee collision)
