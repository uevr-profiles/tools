## Description

This profile is the same as the previous but the map fix is removed because switching to 2d mode intermittently caused crashes. 

Also in this game be careful not to hold back too long to enter the map. You'll end up triggering steam vr and that also causes the game to crash after a few times doing that. I edited the game's ini file for controls and swapped back and start.

Anyone wanting to do the same, the file is InputSettings.ini located in %localappdata%\WB Games\Gotham Knights\Steam folder.  You can use mine below (for gamepad). Just copy it into the folder above.  Note: %localappdata% pasted just like that into windows explorer is a macro for a folder like c:\users\yourname\local and will work just like shown above.
