## Description

6DOF profile including:

- hands / arms (kept the arms because of the cool textures) - I know that some weapons will not be positioned perfectly
- rebinding of X and B, so dodge is now B and X is now interact
- fixed left/right controller swing in main menus
- UI and HUD set to right controller (feel free to change as you please in UEVR settings)

Tip on performance: Set the game to no upscaling. UEVR overlay>Advanced options>Cvars> reduce r.ScreenPercentage as you please

As per Keyser-Soze's suggestion I would also recommend:

Crosshair - Off (you'll still have a dot, but you can ignore it)
Camera Shake - Off
Aim Assist - Off - This one is most important for 6dof
