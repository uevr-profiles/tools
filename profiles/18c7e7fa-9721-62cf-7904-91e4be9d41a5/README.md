## Description

i just spent the day making a lua script that will swap between uncoppled pitch and uses hmd aim when on foot, and copples pitch and locks hud to the windshield when in a car. 

roomscale kindof works, i just don't have the motion controllers configured in a convenent manner. 

havn't tested multiplayer yet but it probably works. 

still got to set FOV to either extream to previent LOD artifacts from popping up, but i hope to automate that at some point if i can figure out how cvars work (if that even is how cvars work) 

use uevr nightly and tweak some ingame settings as follows, 

option>graphics>screen>resolution to 1360x768 and option>gameplay>hud>ui scale to 120% 

or tweak to taste. (sadly tweaking things like position and distance will require diving into the lua file)
