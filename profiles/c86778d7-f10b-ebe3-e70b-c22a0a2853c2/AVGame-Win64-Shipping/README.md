## Description

# First Person Profile
This takes ALL of your performance...decrease graphic settings at least to "High".
## __Features__
- First Person, model invisible
- Dash on RT
- UI nice and close
- sharpened, no depth of field
- Body-Turn and Stick-Turn. When Body-Turn: to catch items always walk towards them a bit. To get Dash-up symbol, turn with stick
## __Issues__
- transp. "shadow model" of the player visible in bright blue lights ("Legacy" visibility - but if turned off, the char would not walk right)
- RT is B (Dash), navigate only with LT in the menus
- some foggy thingies rendered only on one eye (due to Native Stereo) - Sequential solves it, but fps drop by 10
- cursor directions in menus messed up sometimes, solve by untick (and tick again) "UI follows view" in UEVR "Runtime" section
## __Enhancements__
- To eliminate film grain, motion blur and uncap framerate: extract inis.zip (and replace files) into %LOCALAPPDATA%\AVGame\Saved\Config\WindowsNoEditor
