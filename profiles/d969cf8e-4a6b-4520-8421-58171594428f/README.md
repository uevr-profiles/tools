## Description

6 DOF - v1.04 

Features:
- Mod config menu to change from UI/HUD following HMD or right controller aim
- Added VR hands (animated to grip), which should reappear after changing biomes, restarting cycles, - And after cutscenes and interactions (credit: jbusfields libs)
- Attached each weapon with unique hand grips (credit: jbusfields libs), though aim is still ever so slightly to the right of the in-game crosshair
- Included cutscene fix that disables 1st person for cutscenes
- Included Cvars to fix daylight shadows (NB: turn on RT Shadows in game graphics)
- Hidden meshes and objects, including parasites  (credit: markmon, flippy & Azraphyle)
- House sections in first person are still slightly awkward

Other:
- Reverted to no button remapping at all to avoid confusion, as this is available and highly customizable in the game itself (as well as UEVR bindings)
- Short click on left Menu button = Pause
- Longer hold (1-2 seconds) = Open suit computer
- Removed cvars_standard.txt, as users can set their own cvars preferences in UEVR advanced overlay section
